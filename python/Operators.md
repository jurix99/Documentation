# [Operators](https://www.w3schools.com/python/python_operators.asp) - List

### Python Arithmetic Operators
|Operator|Name|Example|
| ------ | ----------- | ------ |
+|Addition|x + y	
-|Subtraction|x - y	
*|Multiplication|x * y	
/|Division|x / y	
%|Modulus|x % y	
**|Exponentiation|x ** y	
//|Floor division|x // y

### Python Assignment Operators
|Operator|Example|Same As|
| ------ | ----------- | ------ |
=|x = 5|x = 5	
+=|x += 3|x = x + 3	
-=|x -= 3|x = x - 3	
*=|x *= 3|x = x * 3	
/=|x /= 3|x = x / 3	
%=|x %= 3|x = x % 3	
//=|x //= 3|x = x // 3	
**=|x **= 3|x = x ** 3	
&=|x &= 3|x = x & 3

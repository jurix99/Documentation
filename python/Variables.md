# [Variables](https://www.w3schools.com/python/python_variables.asp) - Documentation

### Declaration
``` py
a = "string"
b = 1
x = str(3)    # x will be '3'
y = int(3)    # y will be 3
z = float(3)  # z will be 3.0
``` 
### Multiple Declaration
``` py
x, y, z = "Orange", "Banana", "Cherry"
```
### Get Type of a variable
``` py
type(x)
```
